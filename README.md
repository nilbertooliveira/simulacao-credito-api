## Ambiente Docker

1. Clonar o repositório:
  ```
 git clone https://github.com/nilbertooliveira/simulacao-credito-api.git
 composer install
 ```
2. Acessar a pasta do projeto "docker" e rodar o comando:
	`docker-compose up -d`
